## How to Use This Repo

Suggested flow:

1. Start from `1-JAVA Basics`
2. Move topic by topic in order
3. Read notes first
4. Code problems manually in Java
5. Revise after solving questions
6. Maintain a mistakes log while practicing

## Best Use for Placement Prep

This repo works best when used with:

- topic-wise LeetCode / GFG practice
- revision cycles
- mock interviews
- timed problem solving
- handwritten or personal mistake notes

Notes alone are not enough for placement, but notes + coding practice + revision can become a very strong preparation system.
